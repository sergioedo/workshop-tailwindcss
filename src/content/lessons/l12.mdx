## Implementación del modo oscuro

Para activar el modo oscuro en TailwindCSS, configura el archivo `tailwind.config.js`:

```js
module.exports = {
  darkMode: 'class', // o 'media' para detectar la preferencia del sistema
};
```

Esto permite activar el modo oscuro mediante la clase `dark` o de forma automática según la preferencia del sistema.

```html live
<div class="bg-white dark:bg-black text-black dark:text-white">
  Contenido adaptable al modo oscuro
</div>
```

```html live
<button onclick="document.documentElement.classList.toggle('dark')">
  Cambiar Modo
</button>
```

## Uso de clases para modo oscuro

Aquí tienes un ejemplo más complejo que simula un formulario con varias capas y un botón para alternar entre el modo claro y oscuro:

```html live
<div class="bg-gray-100 dark:bg-gray-900 p-8">
  <div class="max-w-lg mx-auto bg-white dark:bg-gray-800 p-6 rounded-lg shadow-lg">
    <h2 class="text-2xl font-bold mb-6 text-gray-900 dark:text-gray-100">
      Registro de usuario
    </h2>
    <form>
      <div class="mb-4">
        <label class="block text-gray-700 dark:text-gray-300 mb-2" for="name">Nombre:</label>
        <input class="w-full px-4 py-2 border border-gray-300 dark:border-gray-600 rounded-lg bg-gray-100 dark:bg-gray-700 text-gray-900 dark:text-gray-200" type="text" id="name" placeholder="Tu nombre">
      </div>
      <div class="mb-4">
        <label class="block text-gray-700 dark:text-gray-300 mb-2" for="email">Correo electrónico:</label>
        <input class="w-full px-4 py-2 border border-gray-300 dark:border-gray-600 rounded-lg bg-gray-100 dark:bg-gray-700 text-gray-900 dark:text-gray-200" type="email" id="email" placeholder="Tu correo">
      </div>
      <div class="mb-6">
        <label class="block text-gray-700 dark:text-gray-300 mb-2" for="password">Contraseña:</label>
        <input class="w-full px-4 py-2 border border-gray-300 dark:border-gray-600 rounded-lg bg-gray-100 dark:bg-gray-700 text-gray-900 dark:text-gray-200" type="password" id="password" placeholder="Tu contraseña">
      </div>
      <div class="flex justify-between items-center">
        <button type="submit" class="px-4 py-2 bg-blue-500 dark:bg-blue-700 text-white font-semibold rounded-lg">
          Registrarse
        </button>
        <button type="button" onclick="document.documentElement.classList.toggle('dark')" class="px-4 py-2 bg-gray-300 dark:bg-gray-600 text-gray-900 dark:text-gray-100 font-semibold rounded-lg">
          Cambiar modo
        </button>
      </div>
    </form>
  </div>
</div>
```
