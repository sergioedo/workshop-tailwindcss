## Uso del plugin Prose

El plugin [@tailwindcss/typography](https://github.com/tailwindlabs/tailwindcss-typography) agrega estilos tipográficos predeterminados a los contenidos en formato HTML largo, como artículos o posts de blogs. Aplica la clase `prose` para obtener automáticamente estilos tipográficos bien formateados.

### Instalación
1. `npm install -D @tailwindcss/typography`

2. Añadirlo en el archivo `tailwind.config.js`:

```js
module.exports = {
  plugins: [require('@tailwindcss/typography')],
};
```

### Ejemplo básico de Prose:

```html live
<div class="prose">
  <h1>Un título grande para el artículo</h1>
  <p>
    Este es un párrafo de ejemplo. El plugin Prose de TailwindCSS te permite estilizar automáticamente el contenido de texto para que tenga un diseño atractivo y fácil de leer.
  </p>
  <ul>
    <li>Punto 1</li>
    <li>Punto 2</li>
    <li>Punto 3</li>
  </ul>
</div>
```

Este plugin es útil cuando necesitas aplicar estilos consistentes en grandes bloques de texto, como en blogs o documentación.

## Uso del plugin Forms

El plugin [@tailwindcss/forms](https://github.com/tailwindlabs/tailwindcss-forms) mejora el estilo de los formularios, ofreciendo clases predeterminadas que mejoran la apariencia de los inputs, selects, textareas, etc. Sin embargo, puedes personalizarlo para manejar formularios más complejos.

### Instalación
1. `npm install -D @tailwindcss/forms`

2. Añadirlo en el archivo `tailwind.config.js`:

```js
module.exports = {
  plugins: [require('@tailwindcss/forms')],
};
```
