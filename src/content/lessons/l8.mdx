## Diseño de layouts flexibles

Flexbox es un modelo de diseño en CSS que permite distribuir elementos en un contenedor, asegurando un alineamiento y un espacio adecuados incluso cuando cambian las dimensiones del contenedor. Con flexbox, podemos organizar elementos horizontal o verticalmente, distribuir el espacio entre ellos, y alinear cada uno según se necesite.

---

## Activar flexbox

```html
<div class="flex">
  <!-- Elementos flexibles -->
</div>
```

---

## Alineación y distribución de elementos

### Alineación horizontal con filas (justificar entre elementos y centrarlos verticalmente)

```html live
<div class="w-[600px] flex flex-row justify-between items-center bg-gray-300 p-8">
  <div class="p-8 bg-red-400 text-white text-lg font-bold">Elemento 1</div>
  <div class="p-8 bg-blue-500 text-white text-lg font-bold">Elemento 2</div>
  <div class="p-8 bg-green-500 text-white text-lg font-bold">Elemento 3</div>
</div>
```

---

### Alineación vertical con columnas (alineación en la parte superior)

```html live
<div class="w-[600px] flex flex-col items-start bg-gray-300 p-8">
  <div class="p-8 bg-red-400 text-white text-lg font-bold">Elemento 1</div>
  <div class="p-8 bg-blue-500 text-white text-lg font-bold">Elemento 2</div>
  <div class="p-8 bg-green-500 text-white text-lg font-bold">Elemento 3</div>
</div>
```

---

## Ejemplo responsive

```html live
<div class="w-full flex flex-col md:flex-row justify-between items-center bg-gray-300 p-8">
  <div class="p-12 bg-red-300 w-full md:w-1/3 text-center text-lg font-semibold"><p>Lorem Ipsum is simply dummy text of the printing and typesetting industry. Lorem Ipsum has been the industry's standard dummy text ever since the 1500s</p></div>
  <div class="p-12 bg-blue-300 w-full md:w-1/3 text-center text-lg font-semibold"><p>Lorem Ipsum is simply dummy text of the printing and typesetting industry. Lorem Ipsum has been the industry's standard dummy text ever since the 1500s</p></div>
  <div class="p-12 bg-green-300 w-full md:w-1/3 text-center text-lg font-semibold"><p>Lorem Ipsum is simply dummy text of the printing and typesetting industry. Lorem Ipsum has been the industry's standard dummy text ever since the 1500s</p></div>
</div>
```

Este ejemplo se adapta al tamaño de pantalla, mostrando los elementos en una columna en pantallas pequeñas y en una fila en pantallas medianas y grandes.
